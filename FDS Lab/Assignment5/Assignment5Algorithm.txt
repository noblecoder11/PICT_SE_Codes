ALGORITHM

Algorithm for INSERTION sort
Initialize iterations = 0
1. Iterate from arr[1] to arr[len(arr)] over the array. 
2. Compare the current element (key) to its predecessor. 
3. If the key element is smaller than its predecessor, compare it to the elements before. Move the greater elements one position up to make space for the swapped element.
4. Increment iterations
5. return iterations

Algorithm for SHELL sort
1. Initialize the value of interval
2. Divide the list into smaller sub-list of equal interval 'interval'
3. Sort these sub-lists using insertion sort
    1. Iterate from arr[interval] to arr[len(arr)] over the array. 
    2. Compare the current element (key) to its predecessor in the sublist i.e. the element 'interval' indices behind the key. 
    3. If the key element is smaller than its predecessor in sublist, compare it to the elements before in the sublist. Move the greater elements 'interval' positions up to make space for the swapped element.
    4. Increment passes
4. Repeat until complete list is sorted
5. Return passes

# Code for insertion sort
DEF insertionSort(arr):
    iterations = 0
    for i in range(1, len(arr)):
 
        key = arr[i]    # Current element under consideration
 
        j = i-1
        while j >= 0 and key < arr[j] :
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
        iterations += 1
    return iterations


# Code for Shell Sort
DEF shellSort(arr):
    n = len(arr)
    interval = n // 2   # initializing interval
    passes = 0
    while interval > 0:
        # sorting elements at 'interval' using insertion sort
        for i in range(interval, n):
            key = arr[i]
            j = i
            while j >= interval and key < arr[j - interval]:
                arr[j] = arr[j - interval]
                j -= interval
            arr[j] = key
        passes += 1
        interval //= 2  #halving interval
    return passes

# Code for getting top 5
DEF topFive(arr):
    shellSort(arr)
    return arr[(len(arr)-5) : len(arr)]

DEF checkArr(arr):
    if (len(arr)<5):
        print("Number of elements should be greater than 5")
        return False
    for i in range(len(arr)):
        if 0<=arr[i]<=100:
            pass
        else:

            return False
    return True
